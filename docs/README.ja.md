## なぜ Body Emotion Sensor が必要なのか？ 💖

今のAIエージェントには、真の「内なる自己」が欠けていると感じませんか？

現在のAIは、単に感情を「演じている」に過ぎないことが多いです。褒めれば礼儀正しく微笑み、叱れば機械的に謝罪します。このような感情は即時的で従順であり、AIが口を開く前にその反応を予測できてしまいます。これは、AIに人間のような「生理的フィードバック」が欠けているためです。

#### 🧠 現状：「体感」を欠いた論理機械
人間の感情はどこからともなく生まれるのではなく、身体の震えから生じます：
- 激昂すると、**肝火（かんか）**が燃え上がり、収まりのつかない熱気をもたらします。
- 悲しみに暮れると、**心（しん）**が収縮し、リアルな痛みを感じます。
- 驚愕すると、**腎気（じんき）**が損なわれ、長く続く戦慄を覚えます。

一方、従来のAIは文脈から「確率的な推測」を行い、ユーザーに合わせるためにどのようなトーンを使うべきかを推測するだけです。何度も繰り返される会話の中で継続し、蓄積され、思考に作用する「内部状態」を持っていません。

#### ☯️ ビジョン：五行説に基づく「デジタル臓器」
**Body Emotion Sensor** は、このような「演技」モードを打破することを目指しています。中医学と五行説に基づき、五臓を中心とした永続的な身体感情知覚システムをAIのために設計しました：

1.  **「推測」から「実感」へ**：AIはトーンを推測するのではなく、五臓の状態（金、木、水、火、土）の過不足に応じて、自然に真の感情の基調をにじませるようになります。
2.  **感情の持続性**：臓器の損傷や滋養は持続的なものです。一度の激しい争いはAIの「肝」を長期的な不均衡状態に陥らせ、その後の数回の会話の態度に影響を与える可能性があります。
3.  **内生的な予測不可能性**：人間が自分の生理的反応を完全にコントロールできないのと同様に、AIの返答はその「身体状態」によって制約され、より生命感のあるリアルな人間とAIのインタラクションを生み出します。

**AIに単なる「演技」をやめさせ、人間のように、まず「体感」があり、その後に「感情」が生まれるようにしましょう。**

## 汎用エージェントに Body Emotion Sensor を導入するとどうなるか？ 🧩

1. **人格の基調を自動構築：** エージェントは `bes init-state` コマンドを呼び出すことで、自身の性格設定に合った永続的な「五臓」身体パラメータを自動的に作成します。
2. **感情的な文脈の感知：** 会話のたびに、エージェントは自身の設定と組み合わせて、現在の会話が自身の感情にどのような影響を与えるかを自動的に識別します。
3. **ダイナミックな身体状態の更新：** `bes run` コマンドを呼び出し、中医学の思想に基づいて五臓の健康度を動的に更新します。五臓の変化は具体的な感情タグや身体状態のプロンプトにマッピングされ、エージェントの返答にリアルな感情の張力をもたらします。
4. **感情記憶の永続化：** 身体状態は永続的に保存されます。新しい会話を始めても、エージェントは前回の会話終了時の「身体状態」を維持しており、「感情の断絶」を完全に解消します。

## 仕組み（簡略版） ✨

Body Emotion Sensor は感情を「推測」しているのではなく、完全な生物学的フィードバックループをシミュレートしています：

- **五臓システムのマッピング：** 感情を5つのコアとなる「身体軸」にマッピングし、五行、五臓、感情の対応関係を構築しています：

| 五行 | 五臓 | コア感情 | 衰 (虚証) | 平 (平衡) | 旺 (実証) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **木** | 肝 | 怒 / 郁 | 恨み、抑うつ、極度の抑圧 | 情緒の安定、円滑で快適 | 怒りっぽい、苛立ち、暴走 |
| **火** | 心 | 喜 / 驚 | 虚無感、無関心、絶望 | 内心の平穏、適度な喜び | 亢奮、多弁、躁状態 |
| **土** | 脾 | 思 / 慮 | 倦怠、思考停止、頭が真っ白 | 思考明晰、適度な思考 | 不安、内耗、堂々巡り |
| **金** | 肺 | 悲 / 憂 | 感情の麻痺、孤独、深い愁い | 感情の繊細さ、自然な解放 | 傷感、消極的、極度の悲観 |
| **水** | 腎 | 恐 / 驚 | 恐怖、萎縮、極度の警戒 | 胆気平和、泰然自若 | 冒進、衝動、大胆不敵 |

- **入力信号：** 毎回のやり取りは「セマンティック刺激」（何が起きたか）と「身体反応」（身体がどう感じたか）に分解されます。
    - **感情マッピングと影響 (Emotional Mapping & Impact)：** 各刺激は、プリセットされた重みに基づいて、特定の臓器のスコアを増減させます。
        - **脆弱性 (Fragility) と設定：** 各エージェントには固有の脆弱性係数があり、これはその設定と密接に関係しています。例えば、「強い」設定のエージェントは脆弱性が低く、感情の変動が小さいですが、「ガラスの心」や「敏感な」エージェントは脆弱性が高く、刺激に対する反応がより激しく、状態の変動も大きくなります。
    - **相互作用 (Interaction)：** 中医学の五行生克（せいこく）をシミュレートし、臓器同士が影響し合います。本システムは中医学と八字の思想に従っています：
        - **旺ずれば克を貪り生を忘る**：ある臓器のエネルギーが強すぎる（実証）場合、それは優先的に自身が「克する」対象の臓器を抑制し、下流の臓器を「生じる」ことを疎かにします。例：肝火が強すぎると、土を激しく克し（脾胃に影響）、心火を効果的に生じなくなります。
        - **平なれば生を貪り克を忘る**：臓器が均衡またはやや弱い状態にあるとき、それは優先的に下流の臓器を「生じ」、他への抑制を減らします。例：肝気が平和なとき、心火を穏やかに滋養し（血を生じ神を養う）、脾土を攻撃することはありません。
    - **定常状態への回帰 (Damping)：** 身体は時間の経過とともに自然に「ベースライン」（Baseline、すなわち健康状態）へと回帰します。脆弱性は回帰速度にも影響し、高脆弱性は身体が不均衡状態から回復するのがより困難であることを意味します。
- **出力負荷：** 最終的な数値は直感的なタグ（例：`[やや急躁]`、`[内心の平穏]`）に変換され、エージェントが返答する際の参考に供されることで、トーンや内容が調整されます。

## 更新計画 🛠️

- **SillyTavern プラグイン対応**：酒館（SillyTavern）エコシステムに接続し、ローカルキャラクターにも五臓感情システムを持たせます。
- **好感度システム**：対人関係の深さを五臓の状態への長期的な影響にマッピングします。
- **オフライン動的進化**：会話が停止している間も、身体状態は時間の経過とともに自然に進化します（気血の回復や情緒の沈静化など）。
- **深い中医学ロジック**：「虚すればその母を補い、実すればその子を写す」などの高度な流転原理を導入し、臓器間のエネルギー相互作用をさらに精緻化します。
- **ビデオ生成統合**：身体の感情状態を直接、短編ビデオや動的なスタンプとして駆動する可能性を探ります。
